Podemos acidionar videos utilizando a tag "video"

dentro da tag "video" abrimos a tag "source" para colocar o arquivo e "track" para adicionar legendas.

"kind" define elementos e da pra colocar peças, "srclang" adiciona legenda. podemos colocar efeitos/funções onde o arquivo pode ampliar na tela também.


Podemos incorporar paginas/conteudos externos para dentro do nosso codigo com a tag "iframe"

o "iframe" é uma evolução do "embased"
